About Zyron

Zexon Development is a private organization, branded as Zexon, that has created projects like Zyron, Zexon Script, and other Roblox Projects. Zexon and Zyron do not collect any data whatsoever. It only connects to Hydrogen and stores data on the user's device.

Zyron is a custom UI for the Roblox Executor named Hydrogen. Zyron is not an executor itself but a layer; it connects to Hydrogen and can execute scripts for you, providing a premium user experience.
